# PRapp
Simple application

## Overview
App containt 3 fragment:
* "Start" fragment to agree term of use or exit from app
* WebView fragment
* When lost internet connection opening "no internet" fragment

Single activity
  
## Stack
* Kotlin
* Dagger 2
* Navigation
* Shared prefs
* View binding

## Connected SDK's
* Firebase
* Facebook
* OneSignal
* ApsFlyer
